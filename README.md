# ⚡ Misión 400

Sistema de alto rendimiento para la preparación del examen ICFES 2026.
Desarrollado para Ingeniería Univalle como plataforma integral de estudio gamificada.

## 🎯 Funcionalidades

- **📚 Seguimiento de Lectura** — Registro diario de páginas leídas con biblioteca personal
- **💪 Racha de Gimnasio** — Disciplina física con rutinas Push/Pull y mini-rutinas de reset mental
- **🎓 Quiz ICFES** — Banco de 30+ preguntas tipo ICFES en Matemáticas, Lectura Crítica, Sociales e Inglés
- **📊 Simulacros** — Registro y análisis de resultados con promedios y mejores puntajes
- **🎯 Misiones Diarias** — 3 misiones aleatorias por día basadas en estilos de aprendizaje multimodal
- **🧠 Salud y Ánimo** — Seguimiento de estado de ánimo, hábitos y bitácora personal
- **⚡ Sistema de XP** — Gamificación completa con experiencia y niveles

## 🚀 Tecnologías

- HTML5 semántico con diseño responsivo
- CSS3 con variables personalizadas, Grid, Flexbox y media queries
- JavaScript vanilla (ES6+) con patrón modular
- localStorage para persistencia de datos
- SHA-256 para verificación de autenticación
